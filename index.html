    Welcome to MVPet's Github Page

    This is the page where all my major projects are hosted. The source is available to download, each project has a demo video showing it off, and most projects are playable in your browser.

    Advance Wars X Fire Emblem

    A turn based SRPG fusing two of Nintendo's best SRPGs. Developed using C++ and SFML API.

    http://mvpet.github.io/FEXAW

    Fighting Game

    A proof of concept fighting game. Was developed using C++ and SFML API.

    http://mvpet.github.io/AGame

    Megaman Battle Chip Challenge Battle Arena

    A game that integrates MOBA elements into Battle Network's gameplay. Was developed using C# and Unity 2D.

    http://mvpet.github.io/MMBNNE/

    A project I recently finished at ITMS at URI. Was developed using C# and Unity3D.

    http://itmsunity.github.io/OceanMovie

    A project I developed for the Nutrition department at URI. Was developed using ActionScript3 and Flixel API.
